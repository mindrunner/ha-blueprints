# ha-blueprints

Home Assistant automations and blueprints for the Lu setup.

## Automations

All automations are exported as JSON from Home Assistant and stored in `automations/`.

| File | Description |
|---|---|
| all_lights_off.json | Turn off all lights |
| bathroom_motion.json | Bathroom motion trigger |
| bed_passive_on.json | Bed passive mode on |
| bedroom_lights_off.json | Bedroom lights off |
| cinema_on/off.json | Cinema mode on/off |
| datacenter_motion.json | Datacenter motion trigger |
| hallway_motion_2.json | Hallway motion trigger |
| living_kitchen_motion.json | Living/kitchen motion trigger |
| lu_home_stop_recording.json | Stop recording when Lu arrives home |
| lu_leaves_turn_on_recording.json | Start recording when Lu leaves |
| office_motion.json | Office motion trigger |
| office_switch.json | Office switch on |
| office_switch_off.json | Office switch off |
| sleep_mode_off_at_11_00.json | Disable sleep mode at 11:00 |
| storage_motion.json | Storage room motion trigger |
| toilet_motion.json | Toilet motion trigger |
